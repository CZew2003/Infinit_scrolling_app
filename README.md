# Infinite Scrolling Images - Atelier Digital Project

## Overview

Welcome to **Infinite Scrolling Images**, an engaging app designed as part of the Atelier Digital course. This application allows users to seamlessly explore an extensive collection of pictures from Unsplash, personalize their search with color filters, and engage with the community through user accounts and reviews.

## Features

### 1. Infinite Scrolling

Enjoy a captivating browsing experience with infinite scrolling. Scroll endlessly through a vast array of stunning images fetched from Unsplash.

### 2. Colorful Filters

Tailor your image search by selecting primary colors using our intuitive color filter feature. Find pictures that match your preferences and style.

### 3. User Accounts

Unlock a personalized experience by creating a user account. Sign in to access additional features and make the most of the app's community-driven functionalities.

### 4. Review and Share

Express your thoughts on your favorite images by leaving reviews. Share your reactions with the community, and discover what others think about the pictures they love.

## Technologies Used

- **Unsplash API:** Seamless integration with Unsplash's API ensures a vast and diverse collection of high-quality images for users to explore.

- **Firebase:** Leveraging Firebase for secure user authentication and efficient data management related to user registration and reviews.
